# Simple Tasbeeh Counter

![C#](https://img.shields.io/badge/Language-C%23-blue)
![.NET](https://img.shields.io/badge/.NET%20Framework-4.7.2-blue)
![WinForms](https://img.shields.io/badge/Platform-Windows%20Forms-blue)

Simple Tasbeeh counter using C# WinForms and .NET Framework 4.7.2

Features  
- ✓ Digital Tasbeeh counter  
- ✓ Timer runs until the Tasbeeh cycle ends  
- ✓ NotifyIcon reminder appears when complete  
- ✓ Clean and minimal interface  
- ✓ Easy to modify and expand for additional options  

How it works  
You press the start button  
The counter begins to run automatically  
The current Tasbeeh phrase is displayed and the counter increases  
When the cycle completes, a concluding message appears  
A notification is displayed to let you know the session has finished
